│   ├── generate_preview_gifs.py       # Gold → animated GIF previews
│   ├── validate_outputs.py            # Validate pipeline outputs
│   ├── decode_one_scenario.py         # Single-scenario decode (proof of concept)

│   └── animate_first_scenario.py      # Animate single scenario
│
├── data/                              # ALL contents gitignored except diagrams/
│   ├── diagrams/                      # Static pipeline PNGs (committed)
│   │   ├── pipeline_simple.png
│   │   └── pipeline_full.png
│   ├── previews/                      # Scenario GIFs — generated locally
│   ├── previews-veritical/            # Alternate GIF orientation — generated locally
│   ├── silver/                        # Parquet: scenarios, tracks, states
│   └── gold/                          # Parquet: all metric tables

├── docs/                              # Developer documentation
│   ├── LOCAL_SETUP.md                 # Step-by-step local setup
│   ├── DATA_PIPELINE.md               # Pipeline explanation

│

---

## What is and is not committed to the repo

### Committed
- All `scripts/` files
- `proto/` (including compiled `_pb2.py` files)
- `data/diagrams/` (static PNGs used by app)
- `docs/`
- `config/`
- `requirements.txt`, `README.md`, `.gitignore`

### NOT committed (gitignored)
- `data/silver/` — parquet files generated by the pipeline
- `data/gold/` — parquet files generated by the pipeline
- `data/previews/` — GIFs generated by `generate_preview_gifs.py`
- `data/previews-veritical/` — alternate GIFs
- `private-dev/` — personal development notes
- `experiments-not-for-github/` — old iterations and scratch scripts
- `backup-local-working-*/` — local backup snapshots
- `.venv/` — Python virtual environment
- `__pycache__/`, `*.pyc` — Python cache files

### Raw Waymo data
Never in the repo. Stored at `~/datasets/waymo/raw/` outside the project directory.
